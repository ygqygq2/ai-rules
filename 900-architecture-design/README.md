## 🛠️ 主要技术概念

### 架构模式

- **分层架构** - 传统的分层设计模式
- **微服务架构** - 分布式服务架构
- **事件驱动架构** - 基于事件的松耦合架构
- **六边形架构** - 端口和适配器模式

### 设计原则

- **SOLID** - 面向对象设计原则
- **DRY** - Don't Repeat Yourself
- **KISS** - Keep It Simple, Stupid
- **YAGNI** - You Aren't Gonna Need It

### 架构工具

- **C4 Model** - 架构图表示法
- **ADR** - 架构决策记录
- **ArchUnit** - 架构规则验证
- **PlantUML** - 架构图绘制
