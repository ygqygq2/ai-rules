## 🛠️ 主要技术栈

### 前端测试

- **Jest** - JavaScript 测试框架
- **React Testing Library** - React 组件测试
- **Cypress** - 端到端测试框架
- **Playwright** - 现代 Web 测试

### 后端测试

- **pytest** - Python 测试框架
- **JUnit** - Java 测试框架
- **Supertest** - Node.js HTTP 测试
- **Testcontainers** - 集成测试容器

### 性能测试

- **k6** - 现代负载测试工具
- **JMeter** - 性能测试平台
- **Artillery** - Node.js 性能测试
