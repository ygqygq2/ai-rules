## 🛠️ 主要技术栈

### 关系型数据库

- **PostgreSQL** - 功能强大的开源关系数据库
- **MySQL** - 流行的 Web 应用数据库
- **SQLite** - 轻量级嵌入式数据库

### 非关系型数据库

- **MongoDB** - 文档型数据库
- **Redis** - 内存数据库和缓存
- **Elasticsearch** - 搜索和分析引擎

### ORM 工具

- **Prisma** - 现代数据库工具包
- **SQLAlchemy** - Python SQL 工具包
- **TypeORM** - TypeScript ORM
