# Skills (技能)

**工作方法论和流程指南**，告诉 AI"如何做事"而非"如何写代码"。

## Skills vs AI Rules

|            | Skills (1300)            | AI Rules (100-900)  |
| ---------- | ------------------------ | ------------------- |
| **关注点** | 工作流程、决策方法       | 技术栈、编码规范    |
| **内容**   | 检查清单、决策树、流程图 | 语法、API、最佳实践 |
| **示例**   | "如何做代码审查"         | "TypeScript 如何写" |
| **跨语言** | ✅ 通用方法论            | ❌ 技术栈特定       |

## 使用场景

**使用 Skills 当你需要**:

- 📋 工作流程指导（如何进行 Code Review）
- 🎯 决策支持（何时重构、如何优化）
- ✅ 检查清单（避免遗漏关键步骤）
- 🔀 流程图（复杂场景的决策路径）

**使用 AI Rules 当你需要**:

- 💻 编码规范（Python 如何写类型提示）
- 🏗️ 架构模式（如何组织 React 组件）
- 🔧 工具配置（如何设置 ESLint）
- 📚 技术细节（API 用法、最佳实践）

## 特点

- ✅ 强调"何时"和"为什么"
- ✅ 提供决策矩阵和检查清单
- ✅ 流程驱动、可操作
- ✅ 语言无关的通用方法论
