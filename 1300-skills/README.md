# Skills (技能)

特定领域的技能和最佳实践集合，用于 AI 辅助开发场景。

## 目录结构

- `1301-python-development.mdc` - Python 开发技能
- `1302-typescript-development.mdc` - TypeScript 开发技能
- `1303-debugging.mdc` - 调试技巧
- `1304-code-review.mdc` - 代码审查技能

## 使用说明

Skills 文件用于配置 AI 助手的特定领域知识，可通过 skills 适配器同步到工作区。

## 特点

- 保持原始格式（包含 frontmatter）
- 可直接用作 AI 提示词
- 支持按领域分类
- 便于独立维护和更新
