## 🛠️ 主要设计工具

### 设计软件

- **Figma** - 协作设计平台
- **Sketch** - macOS 矢量设计工具
- **Adobe XD** - 用户体验设计工具
- **Framer** - 交互原型设计

### 原型工具

- **InVision** - 原型设计和协作
- **Principle** - 交互动画设计
- **ProtoPie** - 高保真原型工具

### 设计系统工具

- **Storybook** - 组件开发环境
- **Zeroheight** - 设计系统文档
- **Abstract** - 设计版本控制

### 用户研究工具

- **Hotjar** - 用户行为分析
- **UserTesting** - 远程可用性测试
- **Maze** - 用户测试平台
